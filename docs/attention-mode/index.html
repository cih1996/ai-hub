传统模式 vs 注意力模式

传统模式

  • 每个会话独立运行进程
  • 空闲时持续消耗资源
  • 智能体之间无法协作
  • 上下文容易丢失

注意力模式

  • 单一进程处理所有会话
  • 空闲会话零资源消耗
  • 智能体可互相发消息
  • 记忆库持久化上下文
